# ✅ Image Position Control - Добавлено в CoverLab

**Дата:** 2026-01-30 14:25  
**Функция:** Контроль позиционирования изображения (как в Tilda)

---

## 🎯 Что было добавлено:

### 1. **Новое поле в интерфейсе**
```typescript
interface CoverData {
    // ... existing fields
    imagePosition: string; // ✅ НОВОЕ
}
```

### 2. **Значение по умолчанию**
```typescript
imagePosition: 'center center' // По центру
```

### 3. **UI Контрол с 9 позициями**

Добавлен визуальный контрол с сеткой 3×3, как в Tilda:

```
┌─────────────┬─────────────┬─────────────┐
│  Left Top   │ Center Top  │  Right Top  │
├─────────────┼─────────────┼─────────────┤
│ Left Center │Center Center│Right Center │
├─────────────┼─────────────┼─────────────┤
│ Left Bottom │Center Bottom│Right Bottom │
└─────────────┴─────────────┴─────────────┘
```

### 4. **Применение к шаблонам**

Изображения в обоих шаблонах (Story и Desktop) теперь используют:
```tsx
<img
    src={data.image}
    alt="Background"
    className="absolute inset-0 w-full h-full object-cover"
    style={{ objectPosition: data.imagePosition }} // ✅ НОВОЕ
/>
```

---

## 🎨 Как использовать:

1. **Загрузите изображение** в Cover Lab
2. **Появится секция "Image Position"** под "Accent Color"
3. **Выберите позицию** из 9 вариантов:
   - **Left Top** - Левый верхний угол
   - **Center Top** - Центр сверху
   - **Right Top** - Правый верхний угол
   - **Left Center** - Левый центр
   - **Center Center** - По центру (по умолчанию)
   - **Right Center** - Правый центр
   - **Left Bottom** - Левый нижний угол
   - **Center Bottom** - Центр снизу
   - **Right Bottom** - Правый нижний угол

4. **Превью обновится мгновенно** в обоих форматах (Story и Desktop)

---

## 💡 Примеры использования:

### Портрет человека:
- Используйте **Center Top** или **Center Center**, чтобы лицо было в центре

### Пейзаж с горизонтом:
- Используйте **Center Bottom**, чтобы показать небо

### Объект слева:
- Используйте **Left Center**, чтобы объект не обрезался

### Объект справа:
- Используйте **Right Center** для правильного кадрирования

## 🔧 Технические детали:

### CSS свойство:
```css
object-position: center center; /* или любая другая позиция */
```

### Поддерживаемые значения:
- `left top`, `center top`, `right top`
- `left center`, `center center`, `right center`
- `left bottom`, `center bottom`, `right bottom`

### Работает с:
- ✅ Instagram Story (1080×1920)
- ✅ Desktop Cover (1920×1080)
- ✅ Сохраняется при скачивании
